# C++ OpenGL Simulator

This repository contains a C++ OpenGL-based simulator designed for testing multi-agent algorithms. The project is currently under active development, and some components may be incomplete or subject to change.

---

## Run App

Click `run.bat`


---

## Current Progress

- **Ackermann Vehicle Model** implemented
- **Rendering System** working (plane displays correctly)
- **Camera System** added:
  - Movement controls:
    - `W / A / S / D` → Move
    - `Shift` → Move down
    - `Space` → Move up
- **Ground Grid** visualization added
